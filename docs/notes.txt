To build the webpages:

   make html


To automatically generate the API docs:

   sphinx-apidoc -f -M -o docs/source/ .

  (-M puts the module before submodule in the API pages)


To support notebooks:

pip3 install nbsphinx --user
dnf install pandoc


For numpy doc format:

pip3 install numpydoc --user


For the theme:

pip install sphinx_rtd_theme
